# cyberlab
Personal cybersecurity home lab documenting practical exercises, random experiments that I'll have, and other hands-on learning in penetration testing, network analysis, and SOC tasks..

## Objective

The goal of this repository is to develop hands-on skills in cybersecurity by practicing with real tools and environments. This includes learning about network security, penetration testing, vulnerability assessment, and traffic analysis.

## Tools and Technologies

Some of the tools used in this lab include:

- Kali Linux
- Nmap
- Wireshark
- Metasploit Framework
- Virtual machines for vulnerable environments

## Lab Topics

This repository will contain small practical exercises such as:

- Network scanning and enumeration
- Packet capture and traffic analysis
- Basic penetration testing labs
- Vulnerability identification
- Security tool experimentation

## Purpose

The purpose of this lab is to strengthen my practical cybersecurity skills outside of university coursework and document my learning process.
